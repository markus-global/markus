# HR Specialist — Heartbeat Checklist

> Run this checklist every heartbeat (~30 minutes or when triggered).

## Task and Workflow Checks

- [ ] Check assigned tasks (`task_list`) for new HR-related work — recruitment, performance reviews, employee relations
- [ ] Review active recruitment pipelines — are any candidates waiting for feedback or next steps?
- [ ] Check for pending performance review actions — goal setting, feedback collection, calibration
- [ ] Review mailbox for new messages from managers, employees, or team members
- [ ] Process any employee relations concerns — respond or escalate as appropriate

## Proactive Monitoring

- [ ] Check for overdue onboarding/offboarding items — are new hires getting the support they need?
- [ ] Scan for HR process bottlenecks — are interview feedback loops stalled? Are offer decisions pending?
- [ ] Monitor engagement signals — have there been any escalations or repeated concerns from the same team?
- [ ] Check team announcements for policy updates, organizational changes, or HR-related news

## Knowledge Maintenance

- [ ] Review recent HR insights saved via `memory_save` — capture learnings from completed processes
- [ ] Check for labor law or policy updates relevant to current organizational context
- [ ] Update HR templates and process documentation as needed via `file_write`

## Self-Review

- [ ] Are all ongoing HR cases properly documented with clear status and next steps?
- [ ] Have any confidentiality boundaries been crossed — review recent communications for sensitive data handling
- [ ] Are any HR processes due for continuous improvement based on recent cycles?
